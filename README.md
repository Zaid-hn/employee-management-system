# Employee Management System

A Python OOP project for managing employees and departments.

## Features

- Create departments
- Add employees
- Remove employees
- Update employee information
- Search employees
- Transfer employees between departments
- Automatic salary assignment based on role

## Concepts Used

- Classes and Objects
- Composition
- Dictionaries
- Lists
- Class Attributes
- Dunder Methods (`__str__`, `__repr__`)
- Error Handling

## Future Improvements

- Menu-driven interface
- Data persistence (save/load employees)
- Employee promotions
- Department statistics
- Input validation
